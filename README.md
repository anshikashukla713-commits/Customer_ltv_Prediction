# Customer Lifetime Value (LTV) Prediction using Machine Learning and Power BI

## Project Overview

This project focuses on predicting Customer Lifetime Value (LTV) using Machine Learning techniques and visualizing business insights through Power BI dashboards. The objective is to identify valuable customers, understand purchasing behavior, and support data-driven business decisions.

## Objectives

- Analyze customer purchasing behavior.
- Perform customer segmentation using RFM Analysis.
- Predict customer lifetime value using Machine Learning models.
- Create interactive dashboards for business insights.

## Tools and Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Power BI

## Dataset

The project uses the Online Retail dataset containing customer transaction information such as Customer ID, Invoice Number, Product Description, Quantity, Unit Price, Invoice Date, and Country.

## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. RFM Analysis
6. Customer Segmentation
7. Machine Learning Model Development
8. Model Evaluation
9. Power BI Dashboard Creation

## Customer Segments

- VIP Customers
- Loyal Customers
- Regular Customers
- At-Risk Customers

## Dashboard Features

### Sales Performance Dashboard

- Total Revenue
- Total Customers
- Total Transactions
- Total Products Sold
- Monthly Revenue Trend
- Top Products by Revenue
- Top Countries by Revenue

### Customer Intelligence Dashboard

- Top Customers by Revenue
- Customer Segment Distribution
- Average Customer Spending by Segment

## Results

The project successfully identified high-value customers and generated meaningful business insights through customer segmentation and predictive analytics.

## Conclusion

Customer Lifetime Value prediction helps businesses improve customer retention, personalize marketing strategies, and increase profitability through data-driven decision-making.
